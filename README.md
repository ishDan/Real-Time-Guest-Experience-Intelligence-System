## 📌 Problem Statement
Hotels often collect guest feedback after checkout, which limits their ability to resolve issues during a guest’s stay. This leads to missed opportunities to improve customer satisfaction in real time.

---

## 💡 Solution
This project builds a data-driven system that analyzes guest reviews to:
- Detect sentiment (Positive, Negative, Neutral)
- Categorize issues (Cleanliness, Staff, Maintenance, Food, etc.)
- Assign a priority score based on severity and urgency

The goal is to help hotel management identify and resolve critical issues proactively.

---

## ⚙️ Tools & Technologies
- Python (Pandas, NLTK - VADER)
- Data Processing & Analysis
- Power BI (Dashboard Visualization)

---

## 🔍 Key Features
- **Sentiment Analysis**: Classifies guest reviews into positive, negative, or neutral
- **Issue Categorization**: Identifies key problem areas such as cleanliness, staff behavior, and maintenance
- **Priority Scoring System**: Combines sentiment, issue type, and urgency to rank problems
- **Interactive Dashboard**: Visualizes insights to support decision-making
